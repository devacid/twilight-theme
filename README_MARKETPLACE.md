# Twilight Theme

Classic Twilight color theme from TextMate, ported for VS Code and Antigravity IDE.

## Quick Start

1. Install the extension
2. Press `Ctrl+K Ctrl+T` (or `Cmd+K Cmd+T` on Mac)
3. Select "Twilight" from the theme list

For full documentation, visit the [GitHub repository](https://github.com/devacid/twilight-theme).
