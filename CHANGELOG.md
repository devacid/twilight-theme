# Changelog

All notable changes to the "Twilight Theme" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-25

### Added
- Initial release of Twilight Theme for VS Code and Antigravity IDE
- Faithful port of the classic TextMate Twilight theme
- Exact color matching from the original theme
- Complete UI theming for all editor elements
- Comprehensive syntax highlighting for 20+ languages
- Git integration colors for file status indicators
- Full terminal ANSI color palette support
- Customization support through VS Code settings

### Color Specifications
- Editor background: `#141414`
- Selection: `#DDF0FF33` (authentic light blue tint)
- Line highlight: `#FFFFFF08` (subtle overlay)
- Invisibles: `#FFFFFF40`

### Syntax Colors
- Comment: `#5F5A60` (italic)
- String: `#8F9D6A`
- Keyword: `#CDA869`
- Constant: `#CF6A4C`
- Entity: `#9B703F`
- Variable: `#7587A6`
- Storage: `#F9EE98`
- Support: `#9B859D`

### Special Scopes
- Embedded source highlighting
- String regexp with special character handling
- C/C++ preprocessor directives
- CSS-specific selectors (tag, class, id, pseudo-class)
- Detailed diff colors for version control
- And many more specialized scopes
